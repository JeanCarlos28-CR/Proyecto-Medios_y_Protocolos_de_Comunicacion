¿Qué es HTTP?

HTTP es el protocolo fundamental que permite la transferencia de información en la World Wide Web. Define cómo se estructuran y transmiten los mensajes entre clientes (navegadores) y servidores.

Fundamento Web

Request → Response

El modelo cliente-servidor funciona mediante solicitudes (Request) que el cliente envía y respuestas (Response) que el servidor retorna. Cada intercambio contiene método, URL, headers y body.

Ciclo completo

Importancia global

HTTP es el pilar de APIs, navegación web, streaming, microservicios y tecnologías cloud. Sin él, la web interactiva no existiría como la conocemos hoy.
